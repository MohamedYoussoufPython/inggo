# Inggo VTC

VTC Moto-taxi à Djibouti

## Installation

```bash
flutter pub get
flutter run
```

## Structure

```
lib/
├── core/          # Design system, services, utils
├── l10n/          # Localisation FR/EN
├── model/         # Modèles Freezed
├── provider/      # Riverpod providers
├── screen/        # Écrans (splash, auth, customer, driver)
└── widget/        # Widgets réutilisables
```
